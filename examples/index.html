JS Int-Range Utility Function

Basic

Default
ints({ from: 1, till: 5 });
Specify Spacing
ints({ from: 1, till: 15, stepsOf: 3 });
Increment || Decrement!
ints({ from: 25, till: 1, stepsOf: 5 });

Advanced

Odd
ints({ from: 1, till: 20, sequence: odd({ stepsOf: 2 }) });
Multiples
ints({ from: 1, till: 35, sequence: multiples({ of: 5, stepsOf: 2 }) });
Custom [& Pluggable] Sequences
ints({ from: 1980, till: 2020, sequence: leapYear({ stepsOf: 2 }) });
